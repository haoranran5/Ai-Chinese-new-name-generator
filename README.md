chinese-name-generator

## 环境依赖

- Node.js 版本：建议 16.x 或更高
- npm（Node 包管理器，随 Node.js 一起安装）

## 安装与运行

1. **克隆项目代码**
   ```bash
   git clone https://github.com/你的用户名/Ai-Chinese-new-name-generator.git
   cd Ai-Chinese-new-name-generator
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **本地运行项目**
   ```bash
   npm run dev
   ```

4. **打包构建**
   ```bash
   npm run build
   ```

5. **预览生产环境构建**
   ```bash
   npm run preview
   ```
